## Minimum Requirements
- Linux Server (Recommended Ubuntu)
- Apache or Nginx
- PHP 5.3 or later
- PHP exec() and proc_open() functions enabled
- PHP GD extension
- FFmpeg and libmp3lame codec
- Youtube-Dl
- Apache's mod_rewrite (Nginx rewriting instructions)
