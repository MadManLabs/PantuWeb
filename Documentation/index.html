Requirements

Minimum Requirements
  • Linux Server (Recommended Ubuntu)
  • Apache or Nginx
  • PHP 5.3 or later
  • PHP exec() and proc_open() functions enabled
  • PHP GD extension
  • FFmpeg and libmp3lame codec
  • Youtube-Dl
  • Apache's mod_rewrite (Nginx rewriting instructions)

Server Configuration

(Note: These instructions are provided as a convenience and courtesy to you. They are not a substitute for a professional server configuration. Generally, it is not my responsibility to teach you how to install and configure packages on, or administer and maintain, a Linux system.)
 
 
Firstly Update Repositories
sudo apt update
 
Step 1: Install Apache
sudo apt-get install apache2
 
Step 2: Install MySQL
sudo apt-get install mysql-server

sudo mysql_secure_installation
 
Step 3: Install PHP
sudo apt-get install php libapache2-mod-php
 
Step 4: Restart Apache
sudo systemctl restart apache2
 
Step 4: Enable modRewrite
sudo a2enmod rewrite
sudo service apache2 restart
sudo nano /etc/apache2/sites-enabled/000-default.conf
add these lines at end 
<Directory /var/www/html>

    AllowOverride All

 </Directory>

sudo service apache2 restart
 
Step 5: Install Phpmyadmin
sudo apt install phpmyadmin php-mbstring php-gettext
sudo phpenmod mbstring
sudo systemctl restart apache2
 
Create dedicated user to access phpMyAdmin dashboard
sudo mysql -u root -p
CREATE USER 'phpmyadminuser'@'localhost' IDENTIFIED BY 'phpmyadminpassword';
GRANT ALL PRIVILEGES ON *.* TO 'phpmyadminuser'@'localhost' WITH GRANT OPTION;
exit
 
Step 6: Give Access Permissions Phpmyadmin
 
sudo nano /etc/apache2/apache2.conf
Then add the following line to the end of the file: 
Include /etc/phpmyadmin/apache.conf
Then restart apache: 
sudo systemctl restart apache2
 
Step 7: Install Youtube-Dl
sudo apt-get install python-pip
sudo pip install youtube-dl
 
Step 8: Install Ffmpeg
sudo apt install ffmpeg

Thats it !

Software Installation

  • Unpack/Unzip the distribution file and upload the contents to a new directory on your web server via FTP.
  • Ensure that the 'downloads folder (in your new directory) have write permissions.
  1. -On Linux, this generally means marking the folders/files with chmod 0777.
 
cd /var/www/html 
sudo chmod 777 downloads
